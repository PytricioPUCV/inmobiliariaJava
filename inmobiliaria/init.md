# Inicializacion

## Requisitos
- Java < 14
- Maven < 3.6.3
- Homebrew (Opcional)


## Instalacion

### Maven

```bash
brew install maven
```

**Nota:** Debes tener instalado Homebrew para poder ejecutar el comando anterior.

### Proyecto

```bash
mvn clean install
```

**Nota:** Debes estar en la raiz del proyecto para poder ejecutar el comando anterior.


## Ejecucion

```bash
mvn exec:java -Dexec.mainClass="App"
```

**Nota:** Debes estar en la raiz del proyecto para poder ejecutar el comando anterior.
